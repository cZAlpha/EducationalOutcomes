# EducationalOutcomes
This project is my capstone project for my BS degree at DSU. It consists of a full stack web application capable of ascertaining educational outcomes of certain courses and majors with respect to their ABET learning outcomes.


# Installing required packages for local development
pip install -r requirements.txt (you may need to change the path depending on where requirements.txt is located)
